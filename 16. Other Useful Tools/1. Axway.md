# Axway SecureTransport (MFT)

## What is Axway?

Axway SecureTransport (ST) is an **Enterprise Managed File Transfer (MFT)** solution.

It is used by banks and large organizations to securely transfer files between:

- Application → Application
- Application → Client
- Client → Server
- Internal Server → External Partner
- External Partner → Internal Server

Instead of using normal FTP, organizations use Axway because it provides:

- Secure file transfer
- Encryption
- Authentication
- Audit logs
- Monitoring
- Scheduling
- Centralized management

---

# What is MFT?

MFT = Managed File Transfer

It is a secure method to exchange business files.

Example:

```
Application A
      |
      |
      V
Axway SecureTransport
      |
      |
      V
Client / Partner
```

# Push vs Pull

## Push Connection

Application sends the file.

```
Application
      |
      |
Push File
      |
      V
Axway
```

Example

Application creates report.csv

Script automatically uploads report.csv to Axway.

---

## Pull Connection

Receiver downloads the file.

```
Application
      ^
      |
Pull File
      |
Axway
```

---

# Internal vs External Connection

## Internal

Communication inside organization.

Example

```
Application A

      |

Application B
```

---

## External

Communication with customer/vendor/client.

Example

```
Company Server

      |

Client Server
```

# Authentication Methods

## Username & Password

Simple authentication.

---

## SSH Key Authentication

Most secure.

Generate

```
ssh-keygen
```

Public Key

```
id_rsa.pub
```

Private Key

```
id_rsa
```

Public key is shared with Axway team.

Private key stays on application server.
