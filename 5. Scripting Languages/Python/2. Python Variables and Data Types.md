# Python Variables and Data Types

Variables are used to store data in memory. Every value stored in a variable has a specific **data type**, which determines the kind of data it can hold and the operations that can be performed on it.

---

# Identifiers (Variable Names)

An **identifier** is the name given to a variable, function, class, or other object in Python.

## Rules for Identifiers

- Must begin with a **letter (A-Z, a-z)** or an **underscore (`_`)**.
- Can contain **letters, digits (0-9), and underscores (`_`)**.
- Cannot start with a digit.
- Cannot contain spaces.
- Cannot contain special characters such as `@`, `#`, `$`, `%`, `&`, etc.
- Cannot use Python **keywords** (reserved words) as identifiers.
- Python identifiers are **case-sensitive**.

> **Note:** `name`, `Name`, and `NAME` are considered three different identifiers.

---

## Valid Identifiers

```python
name = "Natraj"
_age = 23
student1 = "Rahul"
total_marks = 450
```

---

## Invalid Identifiers

```python
2name = "Natraj"     # Starts with a digit
my-name = "Python"   # Hyphen (-) is not allowed
total marks = 90     # Space is not allowed
for = 10             # 'for' is a Python keyword
```

---

## Naming Conventions (Best Practices)

- Use meaningful variable names.
- Separate multiple words using underscores (`snake_case`).
- Use lowercase letters for variable names.
- Avoid single-letter variable names except for temporary use.

### Good Examples

```python
student_name = "Natraj"
employee_age = 25
total_salary = 50000
```

### Poor Examples

```python
a = 25
x = "Natraj"
abc123 = 100
```

---

### Examples

# Data Types

A **data type** defines the type of value stored in a variable.

Python automatically assigns the data type based on the assigned value.

---

# Checking the Data Type

Use the built-in `type()` function to determine the data type of a variable.

## Example

```python
name = "Natraj"
age = 23
price = 25.50

print(type(name))
print(type(age))
print(type(price))
```

**Output**

```text
<class 'str'>
<class 'int'>
<class 'float'>
```

---

# Built-in Data Types

## 1. Integer (`int`)

Stores whole numbers (positive, negative, or zero).

### Examples

```python
a = 25
b = -50
c = 0
```

---

## 2. Float (`float`)

Stores decimal (floating-point) numbers.

### Examples

```python
price = 99.99
pi = 3.14159
temperature = -12.5
```

---

## 3. String (`str`)

Stores text such as words or sentences.

Strings can be enclosed using:

- Double quotes (`""`)
- Single quotes (`''`)
- Triple double quotes (`""" """`)
- Triple single quotes (`''' '''`)

### Examples

```python
name1 = "Python"
name2 = 'Python'
name3 = """Python"""
name4 = '''Python'''
name5 = str("Python")

print(name1, name2, name3, name4, name5)
```

---

## 4. Boolean (`bool`)

Stores only one of two values:

- `True`
- `False`

> **Important:** `True` and `False` must always begin with a capital letter.

### Examples

```python
is_logged_in = True
is_admin = False
```

---

## 5. None (`NoneType`)

Represents the absence of a value or a null value.

### Example

```python
result = None

print(result)
print(type(result))
```

**Output**

```text
None
<class 'NoneType'>
```

---

# Example of Different Data Types

```python
name = "Natraj"
age = 23
price = 499.99
is_student = True
address = None

print(type(name))
print(type(age))
print(type(price))
print(type(is_student))
print(type(address))
```

**Output**

```text
<class 'str'>
<class 'int'>
<class 'float'>
<class 'bool'>
<class 'NoneType'>
```

---

# Summary of Common Data Types

| Data Type  | Description     | Example         |
| ---------- | --------------- | --------------- |
| `int`      | Whole numbers   | `10`, `-5`, `0` |
| `float`    | Decimal numbers | `3.14`, `99.99` |
| `str`      | Text/String     | `"Python"`      |
| `bool`     | Boolean values  | `True`, `False` |
| `NoneType` | No value        | `None`          |

---

# Important Points

- Python is a **dynamically typed language**, so you do not need to declare the data type explicitly.
- The data type is assigned automatically based on the value.
- Use the `type()` function to check the data type of any variable.
- `True`, `False`, and `None` are reserved keywords in Python.
- Variable names should be meaningful and follow the `snake_case` naming convention.

---

# Python Keywords

## What are Keywords?

**Keywords** are reserved words in Python that have a predefined meaning. They are used to define the syntax and structure of the Python language.

> **Important:** Keywords **cannot** be used as variable names, function names, or class names.

---

## Python Keywords List

```text
False      await      else       import      pass
None       break      except     in          raise
True       class      finally    is          return
and        continue   for        lambda      try
as         def        from       nonlocal    while
assert     del        global     not         with
async      elif       if         or          yield
```

> **Note:** The list of keywords may change slightly between Python versions. You can always view the current list using the `keyword` module.

```python
import keyword

print(keyword.kwlist)
```

---

## Using a Keyword as a Variable (Invalid)

```python
for = 10
```

**Output**

```text
SyntaxError: invalid syntax
```

Another example:

```python
class = "Python"
```

**Output**

```text
SyntaxError: invalid syntax
```

---

# Case Sensitivity

Python is a **case-sensitive** programming language.

This means uppercase and lowercase letters are treated as different identifiers.

## Example

```python
name = "Natraj"
Name = "Rahul"
NAME = "Python"

print(name)
print(Name)
print(NAME)
```

**Output**

```text
Natraj
Rahul
Python
```

Here, `name`, `Name`, and `NAME` are three different variables.

---

## Another Example

```python
age = 25
Age = 30

print(age)
print(Age)
```

**Output**

```text
25
30
```

---

## Keywords are Also Case-Sensitive

Python keywords must be written in their correct case.

### Correct

```python
True
False
None
```

### Incorrect

```python
true
false
none
```

**Output**

```text
NameError: name 'true' is not defined
```

# Variables Example

```python
a = 1000
b = 500

diff = a - b

print(diff)
```

**Output**

```text
500
```

### Explanation

- `a` stores the value `1000`.
- `b` stores the value `500`.
- `diff` stores the result of `a - b`.
- `print(diff)` displays the result.
