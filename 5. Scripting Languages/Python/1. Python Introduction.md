# Python Introduction

## What is Python?

Python is a **high-level, interpreted, general-purpose programming language** known for its simplicity and readability. It is one of the most popular programming languages and is widely used in automation, web development, data science, artificial intelligence, machine learning, DevOps, cloud computing, and scripting.

### Key Features

- Simple and easy to learn
- High-level programming language
- Interpreted language
- Free and open source
- Portable (cross-platform)
- Large standard library
- Readable and clean syntax
- python is case sensitive languge ( A and a not same example - Apple and apple is diffrent)

### Developed By

- **Guido van Rossum**
- First released in **1991**

---

# How Python Works

## Machine Language

A computer understands only **Machine Language**, which consists of binary digits:

```text
0 and 1
```

Since humans cannot easily write programs in binary, programming languages like Python are used to communicate with computers.

---

## High-Level Language

Python is a **High-Level Language**, which means:

- Easy for humans to read and write.
- Uses English-like syntax.
- Hides complex hardware operations.
- Makes programming faster and easier.

### Advantages

- Easy to learn
- Easy to debug
- Less code compared to many other languages
- Improves developer productivity

---

## Interpreter

Python is an **Interpreted Language**.

An **Interpreter** reads the program **line by line**, translates it into machine language, and executes it immediately.

### Advantages

- Easy to debug
- Immediate error reporting
- No separate compilation step

> **Interview Note:** Python uses an **Interpreter**, not a Compiler.

---

# Python Character Set

A character set defines the valid characters that can be used while writing a Python program.

Python supports the following character sets:

## 1. Alphabets

```text
A-Z
a-z
```

## 2. Digits

```text
0-9
```

## 3. Whitespace Characters

- Space
- Tab
- New Line

Whitespace is also used for **indentation**, which is mandatory in Python.

## 4. Special Symbols

```text
+  -  *  /  %  =  <  >  ( )  [ ]  { }  ,  :  ;  "  '
```

These symbols are used for mathematical operations, comparisons, assignments, indexing, function calls, and many other programming tasks.

## 5. Unicode Characters

Python supports Unicode, allowing programs to use:

- Multiple languages
- Special symbols
- Emojis

Example

```python
print("नमस्ते")
print("😊")
```

# Python Functions

A **function** is a reusable block of code that performs a specific task.

Python provides many **built-in functions**, and programmers can also create their own functions.

One of the most commonly used built-in functions is `print()`.

---

# print() Function

The `print()` function is used to display output on the screen.

## Printing Text

```python
print("Hello, World!")
```

**Output**

```text
Hello, World!
```

---

## Printing Numbers

```python
print(23)
```

**Output**

```text
23
```

---

## Printing Expressions

Python evaluates expressions before printing them.

```python
print(2 + 5)
```

**Output**

```text
7
```

---

## Printing Multiple Values

Multiple values can be printed in a single `print()` statement using commas.

```python
name = "Natraj"
age = 23

print("Name:", name, "Age:", age)
```

**Output**

```text
Name: Natraj Age: 23
```

---

## Printing on Multiple Lines

Use separate `print()` statements.

```python
print("Python")
print("Programming")
```

**Output**

```text
Python
Programming
```

# Variables

A **variable** is a named memory location used to store data.

Variables allow us to save values and reuse them throughout the program.

## Syntax

```python
variable_name = value
```

---

## Variable Examples

```python
name = "Natraj"
age = 23
price = 25.30
```

Here,

- `name` stores a string.
- `age` stores an integer.
- `price` stores a floating-point number.

---

## Printing Variables

```python
print(name)
print(age)
print(price)
```

**Output**

```text
Natraj
23
25.3
```

---

## Variable vs String

```python
age = 23

print("age")
print(age)
```

**Output**

```text
age
23
```

### Explanation

- `"age"` is a **string**, so Python prints the exact text.
- `age` is a **variable**, so Python prints the stored value.

---

## Assigning One Variable to Another

A variable can store the value of another variable.

```python
age = 23
age2 = age

print(age2)
```

**Output**

```text
23
```

---

## Printing Variables with Text

```python
name = "Natraj"
age = 23
price = 25.30

print("My name is:", name)
print("My age is:", age)
print("The price is:", price)
```

**Output**

```text
My name is: Natraj
My age is: 23
The price is: 25.3
```

# Important Points

- Python is a **high-level programming language**.
- Python is an **interpreted language**.
- Computers understand only **binary language (0 and 1)**.
- `print()` is used to display output.
- Variables store data in memory.
- Strings must be enclosed in quotation marks (`""` or `''`).
- Variable names should not be enclosed in quotation marks.
- Python code is executed line by line by the interpreter.
