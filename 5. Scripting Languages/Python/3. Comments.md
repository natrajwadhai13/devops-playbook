# Comments in Python

## What are Comments?

Comments are used to explain code, improve readability, and make programs easier to understand. Python ignores comments during program execution.

> **Note:** Comments are only for programmers and are **not executed** by Python.

---

# Types of Comments

Python supports two types of comments:

1. Single-Line Comment
2. Multi-Line Comment

---

# 1. Single-Line Comment

A single-line comment starts with the `#` symbol.

## Syntax

```python
# This is a single-line comment
```

## Example

```python
# Print a welcome message
print("Welcome to Python")
```

**Output**

```text
Welcome to Python
```

---

# 2. Multi-Line Comment

Python does not have a dedicated multi-line comment syntax. Developers commonly use **triple quotes** (`''' '''` or `""" """`) to write multi-line comments or documentation.

## Syntax

```python
"""
This is a
multi-line comment.
"""
```

or

```python
'''
This is also
a multi-line comment.
'''
```

## Example

```python
"""
Program Name : Calculator
Author       : Natraj
Version      : 1.0
"""

print("Hello")
```

**Output**

```text
Hello
```

---

# Nested Triple Quotes

Be careful when using triple quotes inside another triple-quoted block.

## Incorrect Example

```python
"""
name = """Python"""
print(name)
"""
```

This will produce a **SyntaxError** because the inner triple quotes close the outer block.

---

## Correct Method

Use different quote styles.

```python
"""
name = '''Python'''
print(name)
"""
```

or

```python
'''
name = """Python"""
print(name)
'''
```

---

# Why Use Comments?

Comments help to:

- Explain complex code.
- Improve code readability.
- Make code easier to maintain.
- Document important information.
- Temporarily disable code during testing (comment out code).

---

# Commenting Code in VS Code

## Comment a Single Line

### Shortcut

**Windows / Linux**

```text
Ctrl + /
```

**macOS**

```text
Cmd + /
```

---

## Comment Multiple Lines

1. Select the lines.
2. Press:

```text
Ctrl + /
```

VS Code automatically comments all selected lines.

---

## Uncomment Multiple Lines

1. Select the commented lines.
2. Press:

```text
Ctrl + /
```

The shortcut toggles comments on and off.

---

# Best Practices

- Write comments only when they add useful information.
- Keep comments short and meaningful.
- Avoid commenting obvious code.
- Update comments whenever the code changes.
- Use comments to explain **why**, not just **what**, the code does.

---

# Common Mistakes

❌ Bad

```python
# Print name
print(name)
```

(The code is already self-explanatory.)

---

✅ Better

```python
# Display the logged-in user's name after successful authentication.
print(name)
```

---

# Important Points

- Comments are ignored by the Python interpreter.
- `#` is used for single-line comments.
- Triple quotes (`'''` or `"""`) are commonly used for multi-line comments or documentation.
- Avoid nesting the same type of triple quotes.
- `Ctrl + /` is the shortcut to comment or uncomment selected lines in VS Code.
