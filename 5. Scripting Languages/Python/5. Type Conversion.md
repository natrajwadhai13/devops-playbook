# Type Conversion in Python

## What is Type Conversion?

**Type Conversion** is the process of converting a value from one data type to another.

Python supports two types of type conversion:

1. Implicit Type Conversion (Automatic)
2. Explicit Type Conversion (Type Casting)

---

# 1. Implicit Type Conversion

Implicit type conversion is performed **automatically by Python** when there is no risk of data loss.

Python converts the smaller data type to a larger compatible data type.

## Example

```python
a = 10        # int
b = 5.5       # float

result = a + b

print(result)
print(type(result))
```

**Output**

```text
15.5
<class 'float'>
```

### Explanation

- `a` is an integer.
- `b` is a float.
- Python automatically converts `a` to `10.0`.
- The result becomes a `float`.

---

## Another Example

```python
x = True
y = 5

print(x + y)
```

**Output**

```text
6
```

> `True` is treated as `1` and `False` as `0` during arithmetic operations.

---

# 2. Explicit Type Conversion (Type Casting)

Explicit type conversion is performed **manually by the programmer** using built-in functions.

This process is also called **Type Casting**.

---

# Common Type Conversion Functions

| Function  | Converts To | Example               |
| --------- | ----------- | --------------------- |
| `int()`   | Integer     | `int("10")`           |
| `float()` | Float       | `float("10.5")`       |
| `str()`   | String      | `str(100)`            |
| `bool()`  | Boolean     | `bool(1)`             |
| `list()`  | List        | `list("Python")`      |
| `tuple()` | Tuple       | `tuple([1,2,3])`      |
| `set()`   | Set         | `set([1,2,2,3])`      |
| `dict()`  | Dictionary  | `dict(name="Natraj")` |

---

# int()

Converts a value to an integer.

## Example

```python
a = "100"

print(int(a))
print(type(int(a)))
```

**Output**

```text
100
<class 'int'>
```

---

# float()

Converts a value to a floating-point number.

## Example

```python
a = "25"

print(float(a))
```

**Output**

```text
25.0
```

---

# str()

Converts a value to a string.

## Example

```python
age = 25

print(str(age))
print(type(str(age)))
```

**Output**

```text
25
<class 'str'>
```

---

# bool()

Converts a value to a Boolean.

## Example

```python
print(bool(1))
print(bool(0))
print(bool(""))
print(bool("Python"))
```

**Output**

```text
True
False
False
True
```

# Practical Examples

## Integer to Float

```python
a = 20

b = float(a)

print(b)
print(type(b))
```

**Output**

```text
20.0
<class 'float'>
```

---

## Float to Integer

```python
price = 99.99

value = int(price)

print(value)
```

**Output**

```text
99
```

> **Note:** The decimal part is removed, not rounded.

---

## Integer to String

```python
age = 25

text = str(age)

print(text)
print(type(text))
```

---

## String to Integer

```python
num = "500"

value = int(num)

print(value + 100)
```

**Output**

```text
600
```

---

# Invalid Type Conversion

Some conversions are not possible and raise an error.

## Example

```python
a = "Python"

print(int(a))
```

**Output**

```text
ValueError: invalid literal for int()
```

---

# Type Conversion with User Input

The `input()` function always returns a **string**.

## Without Conversion

```python
age = input("Enter age: ")

print(type(age))
```

**Output**

```text
<class 'str'>
```

---

## With Conversion

```python
age = int(input("Enter age: "))

print(type(age))
```

**Output**

```text
<class 'int'>
```

# Implicit vs Explicit Type Conversion

| Feature              | Implicit Conversion  | Explicit Conversion  |
| -------------------- | -------------------- | -------------------- |
| Performed By         | Python               | Programmer           |
| Also Called          | Automatic Conversion | Type Casting         |
| Manual Code Required | No                   | Yes                  |
| Safe Conversion      | Yes                  | Depends on the value |

---

# Important Points

- Type conversion changes one data type into another.
- Python performs implicit conversion automatically whenever possible.
- Explicit conversion is done using functions like `int()`, `float()`, `str()`, and `bool()`.
- `input()` always returns a string.
- Converting an invalid string (e.g., `"Python"`) to `int` raises a `ValueError`.
- Converting a float to an integer removes the decimal part.
