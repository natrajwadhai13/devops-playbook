## 📌 Common Names of Symbols

- `~` → **Tilde**

- `&` → **And / Ampersand**
- `!` → **Exclamation mark**
- `*` → **Star / Asterisk**
- `@` → **At the rate**
- `#` → **Hash / Number sign**
- `$` → **Dollar sign**
- `^` → **Caret**
- `_` → **Underscore**
- `-` → **Hyphen / Dash**
