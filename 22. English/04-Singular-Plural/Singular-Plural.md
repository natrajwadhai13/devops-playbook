# Singular and Plural

## What are Singular and Plural Nouns?

A noun can be singular (one) or plural (more than one).

**Hindi:** जब कोई वस्तु, व्यक्ति या चीज़ एक हो तो Singular और एक से अधिक हो तो Plural कहलाती है।

**Marathi:** एखादी व्यक्ती, वस्तू किंवा प्राणी एक असेल तर Singular आणि एकापेक्षा जास्त असतील तर Plural म्हणतात.

---

# Singular Noun

A Singular Noun refers to one person, place, animal, or thing.

### Examples

| Singular | Meaning    |
| -------- | ---------- |
| Book     | एक पुस्तक  |
| Car      | एक कार     |
| Server   | एक सर्व्हर |
| Engineer | एक अभियंता |

### Sentences

- I have a book.
- The server is running.
- He is an engineer.

---

# Plural Noun

A Plural Noun refers to more than one person, place, animal, or thing.

### Examples

| Singular | Plural    |
| -------- | --------- |
| Book     | Books     |
| Car      | Cars      |
| Server   | Servers   |
| Engineer | Engineers |

### Sentences

- I have books.
- The servers are running.
- They are engineers.

---

# Rule 1: Add "s"

Most nouns become plural by adding **s**.

### Examples

| Singular  | Plural     |
| --------- | ---------- |
| Book      | Books      |
| Pen       | Pens       |
| Laptop    | Laptops    |
| Server    | Servers    |
| Container | Containers |

### Sentences

- We have two laptops.
- The servers are healthy.
- Docker containers are running.

---

# Rule 2: Add "es"

If a noun ends with:

- s
- ss
- sh
- ch
- x
- z

Add **es**.

### Examples

| Singular | Plural  |
| -------- | ------- |
| Bus      | Buses   |
| Box      | Boxes   |
| Class    | Classes |
| Watch    | Watches |
| Match    | Matches |

### Sentences

- The buses arrived.
- I bought two watches.

---

# Rule 3: Nouns Ending with "y"

If a noun ends with **y** and before y there is a consonant:

Remove y and add ies.

### Examples

| Singular | Plural    |
| -------- | --------- |
| City     | Cities    |
| Company  | Companies |
| Party    | Parties   |
| Policy   | Policies  |

### Sentences

- Many companies use cloud services.
- Several policies were updated.

---

## Exception

If there is a vowel before y, just add s.

### Examples

| Singular | Plural |
| -------- | ------ |
| Boy      | Boys   |
| Day      | Days   |
| Toy      | Toys   |

---

# Rule 4: Nouns Ending with "f" or "fe"

Change f/fe to ves.

### Examples

| Singular | Plural |
| -------- | ------ |
| Leaf     | Leaves |
| Knife    | Knives |
| Wife     | Wives  |

### Sentences

- The trees have green leaves.
- The chef uses sharp knives.

---

# Irregular Plurals

Some nouns do not follow normal rules.

### Examples

| Singular | Plural   |
| -------- | -------- |
| Man      | Men      |
| Woman    | Women    |
| Child    | Children |
| Person   | People   |
| Foot     | Feet     |
| Tooth    | Teeth    |
| Mouse    | Mice     |

### Sentences

- Many people attended the meeting.
- The children are playing.
- Two men fixed the issue.

---

# Same Singular and Plural

Some nouns remain the same.

### Examples

| Singular | Plural |
| -------- | ------ |
| Sheep    | Sheep  |
| Deer     | Deer   |
| Fish     | Fish   |

### Sentences

- I saw one deer.
- I saw many deer.

---

# Singular vs Plural Verbs

### Singular

Use:

- is
- has
- was

### Examples

- The server is running.
- The engineer has a laptop.

---

### Plural

Use:

- are
- have
- were

### Examples

- The servers are running.
- The engineers have laptops.

---

# Office Examples

### Singular

- The application is slow.
- The manager is available.

### Plural

- The issues are open.
- The applications are slow.

# Key Points

✅ Singular = One

✅ Plural = More than One

✅ Most nouns add "s"

✅ Some nouns add "es"

✅ Words ending with consonant + y become "ies"

✅ Irregular plurals must be memorized

- child → children
- person → people
- man → men
- woman → women

✅ Singular Noun → is / has

✅ Plural Noun → are / have

✅ Very important for Emails, Meetings, Interviews, and DevOps Documentation.
