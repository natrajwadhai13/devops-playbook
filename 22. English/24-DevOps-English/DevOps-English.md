# DevOps English

## Introduction

DevOps English is the English commonly used by:

- DevOps Engineers
- Linux Administrators
- Cloud Engineers
- Site Reliability Engineers (SRE)
- Platform Engineers

This chapter focuses on real-world DevOps communication.

# RCA (Root Cause Analysis) English

---

## Structure

```text id="x2p8m4"
Issue

Impact

Root Cause

Resolution

Preventive Action
```

---

## Example

```text id="j6n4q9"
Issue:
Application downtime.

Impact:
Users could not access the application.

Root Cause:
Database connection failure.

Resolution:
Database service was restarted.

Preventive Action:
Monitoring alerts were improved.
```
