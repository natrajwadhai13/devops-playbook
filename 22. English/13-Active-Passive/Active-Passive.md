# Active Voice and Passive Voice

## What is Voice?

Voice tells us whether the Subject is doing the action or receiving the action.

There are two types of Voice:

1. Active Voice
2. Passive Voice

---

## Hindi Meaning

Voice बताता है कि कार्य कौन कर रहा है और किस पर कार्य हो रहा है।

---

## Marathi Meaning

Voice म्हणजे क्रिया कोण करत आहे आणि क्रियेचा परिणाम कोणावर होत आहे हे दर्शवते.

---

# Active Voice

In Active Voice, the Subject performs the action.

## Formula

```text
Subject + Verb + Object
```

---

## Examples

- Ram writes a letter.
- Natraj deploys the application.
- We completed the task.

---

# Active Voice Structure

### Example

```text
Natraj deployed the application.
```

| Part    | Word        |
| ------- | ----------- |
| Subject | Natraj      |
| Verb    | deployed    |
| Object  | application |

---

# Passive Voice

In Passive Voice, the Object receives the action.

The focus is on the action, not the doer.

---

## Formula

```text
Object + Be Verb + V3 + by + Subject
```

---

## Example

### Active

```text
Natraj deployed the application.
```

### Passive

```text
The application was deployed by Natraj.
```

---

# Why Use Passive Voice?

Passive Voice is used when:

- The action is more important than the doer.
- The doer is unknown.
- Professional writing is required.

---

# Active vs Passive Comparison

| Active Voice               | Passive Voice               |
| -------------------------- | --------------------------- |
| Ram writes a letter.       | A letter is written by Ram. |
| We completed the task.     | The task was completed.     |
| Engineers fixed the issue. | The issue was fixed.        |

# Tense-wise Active to Passive Conversion

---

# 1. Simple Present

## Active

```text
Subject + V1/V1+s + Object
```

Example:

- Ram writes a letter.

---

## Passive

```text
Object + is/am/are + V3
```

Example:

- A letter is written by Ram.

---

# More Examples

### Active

- We deploy applications.

### Passive

- Applications are deployed.

---

# 2. Present Continuous

## Active

```text
Subject + is/am/are + Ving
```

Example:

- We are deploying the application.

---

## Passive

```text
Object + is/am/are being + V3
```

Example:

- The application is being deployed.

---

# 3. Present Perfect

## Active

```text
Subject + has/have + V3
```

Example:

- We have completed the deployment.

---

## Passive

```text
Object + has/have been + V3
```

Example:

- The deployment has been completed.

---

# 4. Simple Past

## Active

```text
Subject + V2
```

Example:

- We deployed the application.

---

## Passive

```text
Object + was/were + V3
```

Example:

- The application was deployed.

---

# 5. Past Continuous

## Active

```text
Subject + was/were + Ving
```

Example:

- We were monitoring the server.

---

## Passive

```text
Object + was/were being + V3
```

Example:

- The server was being monitored.

---

# 6. Past Perfect

## Active

```text
Subject + had + V3
```

Example:

- We had completed testing.

---

## Passive

```text
Object + had been + V3
```

Example:

- Testing had been completed.

---

# 7. Simple Future

## Active

```text
Subject + will + V1
```

Example:

- We will deploy the application.

---

## Passive

```text
Object + will be + V3
```

Example:

- The application will be deployed.

---

# 8. Future Perfect

## Active

```text
Subject + will have + V3
```

Example:

- We will have completed testing.

---

## Passive

```text
Object + will have been + V3
```

Example:

- Testing will have been completed.

---

| -------------- | -------------- |
| I              | Me             |
| We             | Us             |
| He             | Him            |
| She            | Her            |
| They           | Them           |

# Common Passive Voice Patterns

## Is / Are + V3

- The server is monitored.
- The application is deployed.

---

## Was / Were + V3

- The issue was fixed.
- The servers were restarted.

---

## Has Been / Have Been + V3

- The deployment has been completed.
- The changes have been applied.

---

## Will Be + V3

- The release will be deployed.
- The report will be submitted.

# When NOT to Use Passive Voice

Active Voice is usually better for speaking.

### Better

- I completed the task.

### Less Natural

- The task was completed by me.

# Key Points

✅ Active Voice = Subject performs action.

✅ Passive Voice = Object receives action.

✅ Passive Voice uses V3 (Past Participle).

✅ Most Passive Sentences use:

- Is/Are + V3
- Was/Were + V3
- Has Been/Have Been + V3
- Will Be + V3

✅ Passive Voice is very common in:

- Emails
- RCA Reports
- Incident Reports
- Change Requests
- Documentation

✅ Active Voice is usually better for speaking.

✅ Passive Voice is usually better for professional writing.
