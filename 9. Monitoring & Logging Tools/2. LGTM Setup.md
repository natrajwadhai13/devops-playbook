# 4️⃣ Windows Server Collector (Alloy)

Since your app runs on **Windows**, install:

👉 **Grafana Alloy**

It collects:

- OTEL traces
- application logs
- system metrics

---

### Example Alloy config

```
otelcol.receiver.otlp "default" {
  grpc {}
  http {}
}

otelcol.exporter.otlp "gateway" {
  client {
    endpoint = "gateway-alloy:4317"
  }
}

otelcol.service "default" {
  pipelines {
    traces = [otelcol.receiver.otlp.default, otelcol.exporter.otlp.gateway]
    metrics = [otelcol.receiver.otlp.default, otelcol.exporter.otlp.gateway]
    logs   = [otelcol.receiver.otlp.default, otelcol.exporter.otlp.gateway]
  }
}
```

# 4. Install Loki (Logs Storage)

Download

```
wget https://github.com/grafana/loki/releases/latest/download/loki-linux-amd64.zip
unzip loki-linux-amd64.zip
chmod +x loki-linux-amd64
```

Config file

```
nano loki-config.yaml
```

Example

```
auth_enabled: false

server:
  http_listen_port: 3100

common:
  path_prefix: /data/loki

schema_config:
  configs:
    - from: 2024-01-01
      store: boltdb-shipper
      object_store: filesystem
      schema: v13
      index:
        prefix: index_
        period: 24h

storage_config:
  filesystem:
    directory: /data/loki/chunks
```

Start Loki

```
./loki-linux-amd64 -config.file=loki-config.yaml
```

# 8. Linux Alloy Production Configuration

Example **production config**

```
otelcol.receiver.otlp "default" {
  grpc {}
  http {}
}

otelcol.processor.batch "default" {}

otelcol.exporter.otlp "gateway" {
  client {
    endpoint = "gateway-server:4317"
    tls {
      insecure = true
    }
  }
}

otelcol.service "default" {
  pipelines {
    traces  = [otelcol.receiver.otlp.default, otelcol.processor.batch.default, otelcol.exporter.otlp.gateway]
    metrics = [otelcol.receiver.otlp.default, otelcol.processor.batch.default, otelcol.exporter.otlp.gateway]
    logs    = [otelcol.receiver.otlp.default, otelcol.processor.batch.default, otelcol.exporter.otlp.gateway]
  }
}
```

Run

```
alloy run /etc/alloy/config.alloy
```

# 10. Central Alloy Gateway Config

This gateway routes telemetry.

Example

```
otelcol.receiver.otlp "ingest" {
  grpc {}
}

otelcol.exporter.loki "logs" {
  endpoint {
    url = "http://loki-server:3100/loki/api/v1/push"
  }
}

otelcol.exporter.otlp "tempo" {
  client {
    endpoint = "tempo-server:4317"
    tls {
      insecure = true
    }
  }
}

otelcol.exporter.prometheusremotewrite "mimir" {
  endpoint {
    url = "http://mimir-server:9009/api/v1/push"
  }
}

otelcol.service "gateway" {
  pipelines {
    logs    = [otelcol.receiver.otlp.ingest, otelcol.exporter.loki.logs]
    traces  = [otelcol.receiver.otlp.ingest, otelcol.exporter.otlp.tempo]
    metrics = [otelcol.receiver.otlp.ingest, otelcol.exporter.prometheusremotewrite.mimir]
  }
}
```

# 15. Common Problems

| Problem         | Cause                                  |
| --------------- | -------------------------------------- |
| No logs         | Alloy log pipeline missing             |
| No traces       | OTEL instrumentation missing           |
| No metrics      | Prometheus remote write not configured |
| High latency    | batch processor missing                |
| Collector crash | memory limit                           |
