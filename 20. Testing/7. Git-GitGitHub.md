## ğŸ¯ Extra Interview Pointers
- **Difference between Git and GitHub:** Git = tool, GitHub = platform built on Git.  
- **Pull vs. Fetch:** `git pull` = fetch + merge, `git fetch` = only download.  
- **Fork vs. Clone:** Fork = copy repo on GitHub account, Clone = copy repo locally.  
- **CI/CD:** GitHub uses **Actions**, GitLab has **built-in pipelines**.  
- **Open-source vs Enterprise:** GitHub dominates open-source, GitLab strong in enterprise DevOps.
