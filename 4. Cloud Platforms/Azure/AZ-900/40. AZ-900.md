# üü¶ **1. DESCRIBE CLOUD CONCEPTS**

### **1. Which two attributes are characteristics of the private cloud deployment model?**

* Hardware must be purchased
* The company has complete control over physical resources and security

### **2. What are two basic services provided by all cloud providers?**

* compute
* storage

### **3. Which two characteristics are common advantages of cloud computing?**

* geo-distribution
* high availability

### **4. Why is cloud computing often less expensive than on-premises datacenters?**

* You are only billed for what you use

### **5. What is an advantage of cloud computing compared to on-premises deployments?**

* You can scale more quickly

### **6. [Answer choice] refers to upfront costs incurred one time, such as hardware purchases.**

* Capital expenditures

### **7. Which cloud deployment model are you using if you migrate some on-prem servers to the cloud?**

* hybrid cloud

### **8. Increasing compute capacity by adding RAM/CPU to a VM is called:**

* vertical scaling

### **9. In cloud computing, what allows you to deploy apps to regional datacenters worldwide?**

* geo-location

### **10. In a PaaS model, which components are provider-managed?**

* operating system
* physical network

### **11. In which cloud service model does the customer manage the OS?**

* IaaS

### **12. In a SaaS model, what is the customer responsible for?**

* data and access

### **13. Which cloud service model does Microsoft Office 365 use?**

* SaaS

### **14. Which service uses the IaaS model?**

* Azure Virtual Machines

### **15. Which two factors affect Azure costs?**

* resource location
* resource usage

### **16. Which tool estimates the monthly cost of a PaaS-based solution?**

* Azure Pricing Calculator

### **17. How do you minimize VM cost but keep disks and data?**

* Deallocate the VM

### **18. How do you associate resource cost to departments without moving resources?**

* resource tags

### **19. How can you reduce costs for VMs that run continuously?**

* Azure Reservations

# üü© **2. DESCRIBE AZURE ARCHITECTURE & SERVICES**

### **20. Which two components are created in an Azure subscription?**

* resource groups
* resources

### **21. A storage account named *storage001* is an example of:**

* a resource

### **22. Which Azure resource gets separate billing invoices by default?**

* subscriptions

### **23. Which Azure resources can use availability zones?**

* Azure SQL databases
* virtual machines

### **24. What is a software emulation of a physical computer?**

* a virtual machine

### **25. Which compute service deploys identical VM instances?**

* Azure Virtual Machine Scale Sets

### **26. What service executes code in a serverless environment?**

* Azure Functions

### **27. VPN Gateway is used for which scenario?**

* connecting an on-prem datacenter to an Azure virtual network

### **28. What provides Windows apps to Mac/Android users?**

* Azure Virtual Desktop

### **29. Which services allow running applications inside containers?**

* Azure Container Instances
* Azure Kubernetes Service (AKS)

### **30. Which storage service stores unstructured data like images?**

* Azure Blob Storage

### **31. Which two protocols access Azure Files storage?**

* SMB
* NFS

### **32. What is the purpose of defense in depth?**

* multiple security layers to prevent unauthorized access

### **33. What allows a user to manage all resources in a resource group?**

* Azure Role-Based Access Control (RBAC)

### **34. RBAC roles apply to which object?**

* scope

### **35. Which feature requires mobile phone authentication?**

* multi-factor authentication (MFA)

### **36. How do you enforce MFA when users sign in from a specific location?**

* Conditional Access
