## ğŸ“¦ **ARM Templates (Core Topic)**

### **What is an ARM Template?**

* A **JSON file**
* Describes Azure resources declaratively
* Native **Infrastructure as Code (IaC)** for Azure

### **Key Idea**

> â€œDescribe **what** you want, not **how** to create it.â€

---

### **ARM Template Components**

| Component    | Purpose           |
| ------------ | ----------------- |
| `$schema`    | Template version  |
| `parameters` | Input values      |
| `variables`  | Reusable values   |
| `resources`  | Azure resources   |
| `outputs`    | Deployment output |

---

### **Example (Simple ARM Template)**

```json
{
  "$schema": "https://schema.management.azure.com/schemas/2019-04-01/deploymentTemplate.json#",
  "contentVersion": "1.0.0.0",
  "resources": []
}
```
