## â˜ï¸ **What is Azure CLI?**

**Azure CLI (Command-Line Interface)** is a **cross-platform command-line tool** used to **create, manage, and automate Azure resources**.

You can use it on:

* Windows
* Linux
* macOS
* Cloud Shell

It works with **Microsoft Azure** services.

---

## â“ **Why Azure CLI?**

Azure CLI is used because it allows you to manage Azure **faster and more accurately** than the portal.

### **Main Reasons**

* Faster than portal
* Repeatable commands
* Easy automation
* Script-friendly
* Less manual mistakes
